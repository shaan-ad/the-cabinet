How It Works

Three steps to get executive-level perspectives on any decision.

Step 1
Install
Two commands. Takes 30 seconds.
claude plugin marketplace add shaan-ad/the-cabinet
claude plugin install the-cabinet
Step 2
Ask
Point any decision at a specific persona or the full cabinet.
Run this pricing strategy past the CFO
Step 3
Get Perspective
Real frameworks. Sharp questions. Honest pushback on your blind spots.
claude plugin marketplace add shaan-ad/the-cabinet
claude plugin install the-cabinet
  1. Download the plugin zip
  2. In Cowork, go to Customize > + next to Personal plugins
  3. Select Create plugin > Upload plugin
  4. Upload the zip file

Type /plugins in the Claude Code panel, add shaan-ad/the-cabinet as a marketplace source, then install from the GUI.

Every persona is a standalone .md file. Open any file in cabinet/, startup/, or advisors/, copy the content, and paste it as a system prompt in ChatGPT, Gemini, or any AI tool.
